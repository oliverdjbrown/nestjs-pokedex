# Ejecutrar en desarrollo

1. Clone the repository
2. Execute
```
npm install
```
3. You must  have NestJs CLI Installed
```
npm i -g @nestjs/cli
```
4. Run database
```
docker-compose up -d
```

## Stack
* MongoDB
* NestJs
